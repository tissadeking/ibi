  • add intent command format: ibi add intent_type threat duration host(s)

  • delete intent: ibi delete intent_id

  • threat examples: DDoS (ddos_ntp or ddos_dns or ddos_pfcp), DoS-Signaling (dos_sig), API vulnerability (api_vul)
